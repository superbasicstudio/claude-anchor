What Claude Anchor Provides

Persistent Memory

Long-term knowledge that survives across sessions — user preferences, project conventions, important decisions.

Enforceable Rules

Immutable constraints ("NEVER do X") read twice per session to prevent context decay and behavioral drift.

Session Continuity

RAM for crash recovery, short-term memory for multi-session context. Never lose your place again.

Voice & Tone

Control Claude's personality, attitude, vocabulary, and communication style. Loaded first to color everything.

Lessons Learned

A living record of past mistakes and their fixes, so Claude doesn't repeat them across sessions.

Design Preferences

Visual design rules — hover states, WCAG accessibility, typography, flat iconography, UX principles.

npx claude-anchor
